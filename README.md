# Flashcard Forge 🗂️

Веб-приложение для создания и повторения обучающих карточек (интервальное повторение)

---

##  Стек технологий
* **Бэкенд:** Go (Golang)
* **База данных:** PostgreSQL (структура из 10 таблиц)
* **Фронтенд:** HTML, CSS, JavaScript

---

##  Инструкция по локальному запуску

### 1. Настройка базы данных (PostgreSQL)
1. Убедитесь, что у вас установлен и запущен **PostgreSQL**.
2. Создайте пустую базу данных (назовите её, например, `flashcard_db`).
3. Чтобы развернуть структуру таблиц, выполните скрипт `schema.sql` в вашей созданной базе данных.

### 2. Создание файла конфигурации
В **самом корне проекта** (рядом с папками `backend` и `frontend`) создайте файл с именем `.env` (с точкой в начале). Добавьте в него ваши параметры подключения к PostgreSQL:

```env
DB_HOST=localhost
DB_PORT=5432
DB_USER=ваше_имя_пользователя
DB_PASSWORD=ваш_пароль_от_бд
DB_NAME=flashcard_db
SERVER_PORT=8080

Запуск проекта
cd backend
go run .
